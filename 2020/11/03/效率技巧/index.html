效率工作方案合集

发表于 2020-11-03

本文是自己整理的平常工作中用到的效率神器~

1. uTools

uTools是一款极简、插件化、跨平台的现代桌面软件。通过自由选配丰富的插件，打造你得心应手的工具集合。当你熟悉它后，能够为你节约大量时间，让你可以更加专注地改变世界。

插件推荐：json编辑器、在线翻译、图片转文字、剪切板、内网穿透、图床 、todo…

2. 七牛云和picGO设置免费图床

1.下载PIcGo，即展示如下界面：

2.创建七牛云账号—个人认证—新建存储空间

设定AccessKey和SecretKey：点击右上角小头像—密钥管理

设定存储空间名：即新建的存储空间名称。

设定访问网址：空间管理—某个空间详情—空间概览页面

PicGo: 一个用于快速上传图片并获取图片URL链接的工具，内支持多种图床配置。

参考网址：https://www.jianshu.com/p/0997d0d15e55

七牛云的一个月能免费有10G的存储空间，就算超出也价格也非常实惠。

但是有一点，七牛云现在需要自定义域名才能访问，需要完成ICP备案。不然只有30天可用的测试域名进行访问。

3. Typora结合坚果云实时同步笔记（推荐）

我选择Typora作为我最喜欢的编写md笔记的原因在于：

  1. 直接导入md文件的形式，选择一个本地目录，新建笔记，自动在本地目录下创建了md笔记源文件，而不是像其他软件一样是存储在对方服务端的数据库，需要同步（emmm..这个功能可能还需要氪金才能用）。
  2. 图片截图/复制，直接粘贴就可以了，方便实用。
  3. 内有丰富的主题，并且也具备外部css渲染文件的导入主题支持，界面简介美观大方。
  4. markdown原生语法，比如需要放置一些博客或者网上的分享论坛上，如果不是原生语法支持的话，渲染会有问题。
  5. 支持Mac、windows、Linux三大操作系统。

⚠️关于本地文件的同步和丢失问题

——可以结合坚果云实现笔记实时同步。

坚果云是一款便捷、安全的专业网盘产品，通过文件自动同步、共享、备份功能，为用户实现智能文件管理，提供高效办公解决方案。

下载坚果云客户端—创建同步文件夹—将需要实现同步的文件夹拖拽入框中即可。

这样可防止笔记丢失，或者在一台新的电脑上再去坚果云上同步下来就可以啦！

⚠️关于笔记中图片复制粘贴路径问题

1）第一种方式，图片和笔记一起存在本地文件夹中，跟着文件夹一起同步至坚果云。

如果不指定图片路径的话，则会默认存储在应用文件夹下

因此可以打开typora偏好设置-指定存储图片的目录**：（比如以下我指定的是当前笔记所在目录下的效率技巧文件夹中）

🌠注意勾选优先使用相对路径。

因为如果不勾选的话，粘贴图片至笔记中显示的是图片所在的绝对路径，这样就会出现一个问题，在另一台电脑上同步笔记下来的时候，图片显示错误，因为笔记中根据绝对路径就会找不到。

勾选了之后，截图粘贴的时候，就会是以下引用图片的语句。这样无论在哪台机子上，只要同步下来这个笔记文件夹，那么图片都是完美展示。

这个方案呢就是要么笔记和图片一起丢，不然图片永远都是展示正确的。
但是有一点缺憾，如果笔记要上传至博客的话，图片还是需要网络地址。

1）第二种方式，图片结合图床上传至服务器。

这样，一旦在笔记中粘贴图片，则就会自动用PicGo上传至七牛云，并且直接引用七牛云回调的地址用来显示。

比如以下图片的源md语句：

但存在图床也有丢失风险，而且图床的云平台一般是要收取一定费用的。

4. Paste剪切板工具

一次剪切多条数据，可避免重复切换应用来复制多条数据。
操作：正常复制 command + C + 粘贴command + shift + V

场景：

  1. 记录复制的数据、来源。
  2. 实时调用剪切板
  3. 支持智能搜索
  4. 历史记录容量可设置(甚至可以无限)。

  5. 自定义快捷调用板块

    相当与建个文件夹📂，归档。

  6. 同时选择并粘贴多个项目

  7. 可以把复制的文本去除格式变为纯文本。

    可以在设置里设置（默认粘贴为纯文本）

下载地址：https://xclient.info/s/paste-for-mac.html

个人感觉这个[精品MAC应用分享网站]还蛮不错的，里面有很多mac破解版软件，还有一些正版软件限免活动，网站界面绿色美观，省去了很多搜索和下载的复杂链路。

5. Xnip滚动长截图

Mac上方便好用的截图工具，支持窗口滚动截图。

  • 标注
    齐全的标注功能，在标注的同时还可以重新调整截图大小。
  • 滚动截图
    可以让您轻松截取屏幕之外的内容，生成长截图。
  • 窗口截图
    在对窗口进行截取时，Xnip 可以像 Mac 自带的截图工具一样截取窗口的阴影。

6. Snipaste 截图/贴图工具

提升工作效率必备神器！强烈推荐，最好用的肯定是贴图功能。
（有截图功能，也挺好用的，但不支持滚动。贴图功能只要是能复制，就能贴，yyds。）

1.首先，截图功能

  • 智能截图：自动检测界面元素边界。（相当智能）
  • 精准控制截图范围：可用鼠标拖放或者键盘快捷键移动或调整截图框大小，实现截图取景框的像素级精准移动。
  • 截图历史回放

2.屏幕取色

先按下 F1 开始截图，然后对准想要取色的地方，按下键盘 C 键即可获取当前像素点的 RGB 颜色值
！！这个好用

3.然后关键的来了，贴图功能：

无论是本地图片，还是截图结果，甚至于一段复制的文字、代码！都能以贴图窗口的形式置顶显示出来。

这种场景可就丰富了，比如两段代码，你要进行对比，或者作为参考，是不是直接可以复制两段代码，贴图展示就可以了！屏蔽所有的外框，屏幕就不会显得很拥挤了。
比如开会的时候除了主要文档之外，还要展示用到的各种报表、图片、要参考的文字，如果要几个一起说明，是不是要各种切换，那贴图就可以很方便了，将主要文档展示，用到其他什么就直接贴过来就是了~
…

可以对贴图窗口进行各种操作，比如旋转、缩放、设置半透明显示、鼠标穿透等。
它也可以作为文字转图片的功能使用哦~ 无论是纯文本还是HTML格式的富文本，都可以一键转成图片并以贴图指定展示在屏幕上！
