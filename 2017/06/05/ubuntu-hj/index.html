‘default_encoding = sys.getdefaultencoding()’。

在这行前面添加三行：

mkdir ~/workspaces //创建一个文件夹，用于存放所有的虚拟环境（也可不创建）

//设置环境变量，把上面两行添加到~/.bashrc里
